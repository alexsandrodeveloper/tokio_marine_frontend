# tokio_marine_frontend

### Configuração do Frontend

1. Navegue até o diretório do frontend:

	```bash
	cd frontend

2. Instale as dependências do Angular:
	```bash
	npm install
	
3. Inicie o servidor de desenvolvimento:
	```bash
	ng serve

4. Abra o navegador e acesse http://localhost:4200 para visualizar a aplicação frontend

### Acesso às Funcionalidades

Para acessar as funcionalidades da aplicação, siga os passos abaixo:

1. Registro de Usuário:

	1.1 Primeiro, crie um usuário utilizando o menu de Registro no frontend Angular.
	
2. Autenticação:

	2.1 Após registrar o usuário, vá para o menu de Autenticação no frontend.
	2.2 Autentique-se utilizando o usuário e senha cadastrados para obter o token de acesso ao backend.
